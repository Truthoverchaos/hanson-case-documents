# Topics Index — Cross-Cutting Substantive Themes

**Purpose:** One file per major substantive theme that cuts across multiple cases. Each topic file:
- Frames the theme as it would appear in the federal complaint
- Pulls evidence from all relevant cases (with KB cross-references)
- Tags the supporting ledger fact IDs
- Identifies what's verified vs. what still needs source verification

When drafting a federal-complaint section or a post on a specific theme, start with the topic file — it pre-organizes the facts.
